# AGENTS.md

## Setup commands

- Install deps: `npm install`
- Start dev server: `npm run dev`
- Build for production: `npm run build`
- Run tests: `npm test`

## Project overview

This is a Next.js application with a modular agent system. Agents are autonomous components that can perform tasks, make decisions, and interact with external services.

## Code style

- TypeScript strict mode enabled
- Use functional components and hooks
- Components in `components/`, UI components in `components/ui/`
